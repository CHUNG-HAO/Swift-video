# Swift-video
> 匯入

> AVKit:

這是Apple的一種用於播放音樂和影片的框架

> MobileCoreServices:

這是Apple的一種提供各種核心服務的框架，文件處理

> @State: 

這是SwiftUI的一種屬性包裝器

> View: 

這是SwiftUI中的一種協議，所有的用戶界面元素都需要遵循這個協議。

> UserDefaults: 

這是一種用於存儲用戶的默認設置的連接

> FileManager:

這是一種用於管理和操作文件和目錄的接口。用來保存影片文件。

> NotificationCenter:

這是一種用於發送和接收應用程序級別通知的連接。它被用來處理影片播放結束的通知。
